## Getting Started
1. Clone the repo & install dependencies:
   ```bash
   git clone https://github.com/John-Jepsen/dsa-quiz-master.git
   cd dsa-quiz-master
   npm install
   ```
2. Start the dev server:
   ```bash
   npm run dev
   ```
3. Open [http://localhost:5173](http://localhost:5173)
